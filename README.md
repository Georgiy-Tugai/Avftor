Avftor Text Editor
------------------

Build Instructions
==================

`qmake`:

    cd build
    qmake ..
    make

`cmake`:

    cd build
    cmake ..
    make

To run:

    ./avftor


Name Info
=========

Name: Avftor
Acronym Expansion: Avftor: Very Fast, Tidy, Open, Reconfigurable
Short form: Avf, avf
